## Dependencias requeridas

- PipeWire (alsa, pulse, jack)
- wireplumber
- pavucontrol
- power-profiles-daemon
- auto-cpufreq
- networkmanager + applet
- nm-connection-editor
- netctl
- hyprland, hypridle, hyprlock, hyprpaper
- waybar, wofi, kitty
- thunar, mousepad
- nwg-look, mate-polkit
- wlogout, swaync
